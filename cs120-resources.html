CS 120 Resources

CS Lab

The Department of Computer Science operates a computer lab for students in CS courses in Wing Technology Center 16. The computers in that room are setup with the tools you need for the programming assignments in CS 120. You may also take your laptop if you prefer.

Lab Hours

Regular lab hours will begin on Sunday September 9. The schedule will be posted here when available.

Study Sessions

Several times throughout the week, the CS Lab is staffed with students who are available to help you with assignments in CS 120. This is a valuable resource, please be sure to take advantage of it if you are struggling with an assignment or topics in the course. You can also go to the lab at these times even if you don't have questions. The schedule will be posted here when avaiable.

Installing Java

Java is available for free in the form of the JDK -- Java Development Kit. You can download the JDK from Oracle. Be sure to click the download button for JDK, NOT the JRE. The newest version of JDK (as of September 3, 2018) is 10.0.2.

Installing Eclipse

There are many tools that will allow you to edit Java code. While you are welcome to use any of them on your computer, in lab we will use Eclipse. Eclipse is a free Integrated Development Environment (IDE). IDEs allow you to write and run code from within a single program. The current version is Eclipse Photon, which can be downloaded from the developer, Eclipse.org. Installation is painless.

Using Eclipse

The department has a set of tutorials to help familiarize you with Eclipse. Though they were not created for the most recent version, they should be very helpful. You can find them here. Thanks to Dr. Josh Hursey for sharing these.
